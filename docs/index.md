# 字体库

### 字体包下载地址

[字体天下](https://www.fonts.net.cn/)
[阿里巴巴字体库](https://www.iconfont.cn/fonts/index?spm=a313x.home_index.1998910419.16.58a33a81N6ZzSO)

### 引用

```style
@font-face {
    font-family:'字体库名称'
    src: 字体包地址，local()函数本地字体包，url(网络字体包)
}
```
